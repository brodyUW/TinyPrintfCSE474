Print formatted string to UART TX.

printf_t is a user-accessible wrapper function for performing lightweight printf-style output to a serial console. Specific format functioning is performed in the formatCommon function. Usage: printf_t("Hex value[%u] = 0x%x\n", var1, var2);

Parameters
[in]*formatCharacters to be sent to UART TX buffer.
[in]...Variadic arguments to be printed.
Returns
void
Note
Be aware that in Arduino IDE, printf and sprintf are already sourced. Be sure to use printf_t or sprintf_t!
